# Dossiê Técnico: Terra Plana, Bíblia, Firmamento e Gigantes

Site estático em HTML, CSS e JavaScript sobre argumentos terraplanistas bíblico-cosmológicos, firmamento, hebraico bíblico, gigantes/nefilim, relatos históricos/modernos e refutações científicas replicáveis.

## Arquivos
- `index.html` — página principal do dossiê.
- `styles.css` — estilos responsivos com estética limpa/moderna e toque vintage.
- `script.js` — menu mobile e busca interna.

## Como usar
Abra `index.html` em qualquer navegador moderno. Para publicar, envie a pasta completa para um servidor estático ou GitHub Pages.
